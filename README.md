# Qwuen Lizzy's Stock Prediction

A web application that provides stock predictions and investment advice based on historical stock data over the past three days. The app fetches real-time stock information using the Polygon API and leverages OpenAI's GPT-4o-mini model to generate personalized buy, hold, or sell recommendations.

## Features

- **Stock Ticker Input**: Add multiple stock tickers (e.g., MSFT, TSLA) to analyze.
- **Historical Data Fetching**: Retrieves stock price data for the past 3 days using Polygon API.
- **AI-Powered Analysis**: Uses OpenAI to generate concise reports (<150 words) with trading recommendations.
- **User-Friendly Interface**: Simple HTML/CSS/JavaScript frontend with a clean, responsive design.
- **Real-Time Reports**: Generates and displays reports dynamically after data processing.
- **Disclaimer**: Clearly states that this is not real financial advice.

## Tech Stack

### Frontend
- **HTML5**: Structure and content.
- **CSS3**: Styling and layout.
- **JavaScript (ES6 Modules)**: Client-side logic, API calls, and DOM manipulation.
- **Vite**: Build tool and development server.

### Backend
- **Node.js**: Runtime environment.
- **Express.js**: Web framework for API endpoints.
- **OpenAI API**: For generating stock analysis reports.
- **Polygon API**: For fetching historical stock data.
- **CORS**: Handling cross-origin requests.
- **Dotenv**: Environment variable management.

### Utilities
- **Dates Utility**: Custom module for calculating date ranges (past 3 days).

## Installation

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn
- API Keys:
  - Polygon API Key (for stock data)
  - OpenAI API Key (for AI analysis)

### Steps

1. **Clone the Repository**:
   ```bash
   git clone <repository-url>
   cd qwuen-sprediction
   ```

2. **Backend Setup**:
   - Navigate to the backend directory:
     ```bash
     cd backend
     ```
   - Install dependencies:
     ```bash
     npm install
     ```
   - Create a `.env` file in the `backend` directory and add your API keys:
     ```
     OPENAI_API_KEY=your_openai_api_key_here
     ```
   - Start the backend server:
     ```bash
     npm run dev
     ```
     The server will run on `http://localhost:3000`.

3. **Frontend Setup**:
   - Navigate to the frontend directory:
     ```bash
     cd ../frontend
     ```
   - Install dependencies:
     ```bash
     npm install
     ```
   - Create a `.env` file in the `frontend` directory and add your Polygon API key:
     ```
     VITE_API_KEY=your_polygon_api_key_here
     ```
   - Start the frontend development server:
     ```bash
     npm run dev
     ```
     The app will be available at `http://localhost:5173`.

## Usage

1. Open the application in your browser at `http://localhost:5173`.
2. Enter stock tickers (e.g., MSFT for Microsoft) in the input field.
3. Click the "Add" button or press Enter to add tickers.
4. Once you've added at least one ticker, click "Generate Report".
5. Wait for the app to fetch data and generate the AI-powered report.
6. View the report in the designated section, which includes performance analysis and recommendations.

### Example
- Input: MSFT, AAPL
- Output: A report summarizing the stock performance over the past 3 days and advising on buy/hold/sell actions.
- To see a screenshot of the output go to /frontend/images/screenchot(103).png

## API Endpoints

### Backend
- `POST /fetchReport`: Accepts stock data and returns an AI-generated report.
  - Request Body: `{ "stockData": "stringified stock data" }`
  - Response: `{ "report": "AI-generated report text" }`
